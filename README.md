# TRM
TRM assignment 
1.	Look at a website and figure out if it's real or fake
2.	Find hidden clues like cryptocurrency wallet addresses
3.	Connect the dots between different scam websites run by the same criminals
4.	Give you a report saying 'SCAM' or 'NOT_SCAM' with reasons why
5.  MATHEMATICAL METHODS implemented:
